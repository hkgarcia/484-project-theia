## Installation

To get a local copy up and running follow these simple steps.

### Prerequisites

- Install dependencies

   ```bash
   npm install
   ```

### Running the App

1. Start the app

   ```bash
   cd src
   npx expo start
   ```

2. Ensure **Expo Go** is in installed on your mobile device. Scan the QR code from the terminal or use an iOS simulator (XCode on desktop) to view the app.
